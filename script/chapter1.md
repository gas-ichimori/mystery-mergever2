# 第一章 脚本

## ゲームあらすじ

都心に探偵事務所を構えた新米探偵。
助手のヤスと一緒に、探偵事務所を切り盛りすることになる。

依頼内容は、動物を探すことから浮気・不倫現場の証拠を掴むことまで様々。
そんな探偵事務所には、不可解な依頼から思いもよらない事件にも巻き込まれることもある。

日常的なことから非日常的な物語を、"あなた"は体験することになる。

---

## 第一章あらすじ

依頼人ミユから「猫のミケが居なくなった」と相談を受ける。
調査を進めるうち、猫が現れた場所・荷物の謎・配達業者の不審な動きが絡み合い、
単純な猫探しが思わぬ事件へと発展していく。

## 演出メモ
| 演出タイプ | 説明 |
|------------|------|
| スライドイン | キャラクターが画面外からスライドして登場 |
| フェードイン | キャラクターがじわっと表示 |
| 暗転 | 話していないキャラクターは暗く表示 |
| hideAll | 全キャラ消去・背景も黒に |
| changeBg | 背景を切り替える（キャラ消去と同時） |
| setBg | 背景だけ切り替える（キャラは維持） |
| showCenter | 画面中央にアイテム画像をフェードイン表示 |
| shakeCenter | 中央アイテムを振動アニメーション |
| 左右反転 | ヤスが右向きに変換して表示 |
| shiftRight | 右キャラを左にずらす（3人目登場時） |
