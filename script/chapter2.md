# 第二章 脚本

## ストーリー概要

マンションで不審な嫌がらせが続いているとリナが相談に来る。
調査を進めると、マンション管理会社のジンが立ち退き工作を仕掛けていることが判明。
しかしジン自身も、会社の不正を知り上司に脅されており——ヤスに「解決策がある」と唆された被害者だった。

**第一章 Scene09 と並走開始。**

| 暗転 | 話していないキャラクターは暗く表示 |
| hideAll | 全キャラ消去・背景も黒に |
| changeBg | 背景を切り替える（キャラ消去と同時） |
| showCenter | 画面中央にアイテム画像をフェードイン表示 |
| shiftRight | 右キャラを左にずらす（3人目登場時） |
